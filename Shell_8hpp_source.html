162  static constexpr Options::String help{
163  "Creates a 3D spherical shell with 6 Blocks. `UseEquiangularMap` has\n"
164  "a default value of `true` because there is no central Block in this\n"
165  "domain. Equidistant coordinates are best suited to Blocks with\n"
166  "Cartesian grids. However, the option is allowed for testing "
167  "purposes. The `aspect_ratio` moves grid points on the shell towards\n"
168  "the equator for values greater than 1.0, and towards the poles for\n"
169  "positive values less than 1.0. The user may also choose to use only a "
170  "single wedge (along the -x direction), or four wedges along the x-y "
171  "plane using the `WhichWedges` option. Using the RadialPartitioning "
172  "option, a user may set the locations of boundaries of radial "
173  "partitions, each of which will have the grid points and refinement "
174  "specified from the previous options. The RadialDistribution option "
175  "specifies whether the radial grid points are distributed linearly or "
176  "logarithmically for each radial partition. Therefore, there must be N+1 "
177  "radial distributions specified for N radial partitions. For simple "
178  "h-refinement where the number but not the locations of the radial "
179  "boundaries are important, the InitialRefinement option should be used "
180  "instead of RadialPartitioning."};
